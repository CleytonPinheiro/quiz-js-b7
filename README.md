# QUIZ - Perguntas e respostas -  JAVASCRIPT - ES6

- Jogo de perguntas e respostas, apenas uma das 4 alternativas é a correta.

## Stack utilizada

**Front-end:** HTML, Javascript e CSS.

- Resumo fina, com dados sobre ao aproveitamento da rodada.
